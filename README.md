# 🏥 Healthcare Appointment Booking System

A modern web application built with **Next.js** and powered by **Appwrite**, designed to provide a seamless and efficient way for patients to book and manage healthcare appointments online.  

This system simplifies the traditional fragmented process by offering a **unified platform** where patients can:
- Access a doctor's schedule  
- View medical records  
- Book appointments in one place  

---

## ✨ Inspiration
The project is inspired by the need to address inefficiencies in traditional healthcare administration. By centralizing patient data and appointment scheduling, this platform aims to:
- Improve patient experience  
- Reduce administrative overhead for clinics  
- Promote better, more coordinated care  

---

## 🚀 Features
- **Intuitive Appointment Booking**: A user-friendly interface to quickly find and book appointments.  
- **Unified Patient Records**: Securely centralize medical history, test results, and prescriptions.  
- **Provider Management**: Dashboard for healthcare providers to manage schedules and patient visits.  
- **Secure Authentication**: Powered by Appwrite's robust authentication system.  
- **Global Deployment**: Hosted on **Appwrite Sites** for fast, secure, and scalable performance.  

---

## 🛠 Tech Stack
- **Frontend**: Next.js, React, JavaScript/TypeScript  
- **Backend**: Appwrite (Authentication, Database, Storage)  
- **Deployment**: Appwrite Sites
